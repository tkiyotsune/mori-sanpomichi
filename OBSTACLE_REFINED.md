# 障害物デザイン改善ガイド

## 1. 現状の課題整理

### 切り株（最優先）
- **幹の側面色が `#E8DEC8`（クリーム白）** → 周りの緑の苔と相まって全体が「緑がかった薄い」印象
- 上面と側面の色差が弱く、立体の切り分けが曖昧
- 白樺風の色を使っていて「切り株」の素朴な存在感が弱い

### 湖
- 形が完璧な楕円で「コピペ感」が強い
- 水色がくすみすぎていて、水面のきらめきが弱い
- 情報量不足で印象に残らない

### 岩
- 灰色のベタ塗りで「ただの塊」
- 面の切り替えが乏しく、立体が弱い
- 苔・ベリーの配置が機械的

---

## 2. 改善方針

### 2-1. 共通ルール
```
輪郭線:     基本なし（例外: ごく薄い濃グレー stroke opacity 0.3）
塗り:        BASE + DARK（左下半分）の2層フラット
ハイライト:  最小限、1〜2箇所のみ
接地影:      2段重ね（大: opacity 0.06 + 小: opacity 0.12）
色数:        障害物1種類あたり最大5色
```

### 2-2. 形のゆらぎ
```
完璧な円・楕円は使わない
左右非対称の制御点を1〜2ユニットずらす
上下も微妙にずらす（重力を感じさせる）
```

## 4. 形状仕様書

### 4-1. 岩
```
シルエット:
    _____
   /     \
  /       \___
 /            \
|  rock       |
 \___________/
   (shadow)

- 上部は緩やかな山型（左より右が少し高い）
- 底面は広く水平に近い
- 左下に影レイヤー
- 苔は左上と右下に小さく
- 色点（ベリー）は2個まで
```

### 4-2. 切り株（木材色の統一）
```
シルエット:
   ====       ← 上面（苔つき）
  |====|      ← 年輪（上面のみ）
  |    |      ← 側面：木材色で統一
  | wood |    ← 樹皮線を数本
  |    |
  |____|
  (shadow)

- 上面: 緑の苔 + 茶系の年輪
- 側面: warm tan（#A88050）統一
- 樹皮マーク: 暗い茶（#6A4520）の小さな楕円3〜4個
- 左半分に DARK レイヤー
- 緑は絶対に側面に入れない
```

### 4-3. 湖
```
シルエット:
    ___
   /   \___
  /        \
 |  water   |
  \___    _/
      \__/

- 楕円ベースだが上下で幅を変える
- 波紋: 斜めの弧を3本（色違い）
- 睡蓮: 2箇所に非対称配置
- 水面ハイライト: 楕円の上1/3にうっすら明色の帯
- 浮いた花びら: 3〜4個
```

## 5. SVG実装ルール

### 構造
```
<Svg>
  {/* 1. Ground shadow (2 layers) */}
  {/* 2. Base shape (BASE fill) */}
  {/* 3. Shadow layer (DARK fill, left-bottom half) */}
  {/* 4. Highlight (LIGHT, subtle) */}
  {/* 5. Details (moss, rings, ripples, etc) */}
  {/* 6. Small accents (berries, flowers, leaves) */}
</Svg>
```

### パーツ数の目安
```
岩:      10〜12 paths
切り株:  12〜14 paths
湖:      14〜16 paths
```

### 左右非対称の入れ方
```
- ベース形状の左右制御点を0.5〜1ユニットずらす
- 上面と底面の高さも0.5ユニット程度ずらす
- 苔や装飾は完全非対称に配置
```

---

## 6. 実装優先度

### Phase 1（必須）
1. **切り株の幹を木材色に統一**（#A88050系）
2. 切り株の上面 = 緑苔のみ、側面 = 茶色のみ
3. 樹皮マークを暗茶色（#6A4520）に

### Phase 2
4. 岩の形を非対称に、2層シェーディング
5. 湖の楕円にゆらぎ、水面ハイライト帯
6. 接地影の2段重ね統一

### Phase 3
7. 装飾の上品化（苔・ベリー・睡蓮）
8. 色のくすみを統一

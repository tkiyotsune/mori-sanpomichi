アプリについて

「もりのさんぽみち」は、森の中をのんびり歩くシンプルな癒やしゲームです。
タップだけの簡単操作で、どなたでもお楽しみいただけます。

特徴

  • タップするだけのシンプル操作
  • やさしい色合いの森の世界
  • すきま時間にぴったりの気軽なプレイ
  • オフラインでも遊べます

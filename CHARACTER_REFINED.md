# キャラクター洗練ガイド v2

## 1. 現状の問題点整理

### 1-1. 「素材集感」の正体
現状4体とも以下が共通している:
- 頭の半径 11〜12
- 体の高さ y=25〜44
- 脚の Ellipse サイズが同じ
- 目の位置・サイズが同じ
- 影のレイヤー構造が同じ

→ **テンプレを流し込んだだけ**に見える。各キャラの個性が「アクセサリーの違い」だけ。

### 1-2. フラットすぎて安っぽい
- 塗り2層（ベース + 片側シャドウ）だけで、量感が足りない
- ハイライトがない
- 目にハイライトがなく無機質

### 1-3. シルエットの差別化不足
- 4体とも「丸頭 + 縦長胴 + 小さな脚」で上半身のシルエットが同じ
- 動物の特徴パーツ（尻尾、角、耳）が小さすぎる

### 1-4. 左右対称すぎる
- 目・耳・脚が完全対称 → マスク感
- 「手作り感」がゼロ

## 2. キャラクター別アイデンティティ

### 2-1. 小人 (Kobito) — **「賢者」**
```
キャラ性:  落ち着き、温かさ、静けさ
体型:      ずんぐり、低重心
シルエット: 背の高い帽子 + ひげが顔の大部分を占める
誇張:      帽子を体と同じ高さに、ひげを広く豊かに
特徴:      鼻だけが見える、目はひげの上に小さく
```

### 2-2. きつね (Kitsune) — **「遊び人」**
```
キャラ性:  活発、いたずら、好奇心
体型:      スリム、すらり
シルエット: 体より大きい尻尾 + 尖った耳
誇張:      尻尾を体の1.5倍の大きさに、S字カーブ
特徴:      目のまわりにオレンジの模様（アイマスク的に）
```

### 2-3. くま (Kuma) — **「優しい大きな子」**
```
キャラ性:  のんびり、優しい、まっすぐ
体型:      ぽっちゃり、横幅広め
シルエット: 丸い耳 + 丸い顔 + ずんぐりした丸い体
誇張:      すべてが丸、コロンとしたフォルム
特徴:      マフラーの先が横に流れて動きを出す
```

### 2-4. しか (Shika) — **「気高い」**
```
キャラ性:  優雅、静か、好奇心
体型:      すらり、縦長、脚が長め
シルエット: 枝分かれする角 + 細い首 + 細長い体
誇張:      角を大きく、脚を長く、首を細く
特徴:      背中の白い斑点模様
```

## 3. 比率再設計（4体ごとに変える）

### 従来（全キャラ共通で使っていた）
```
頭:   r=11〜12
体:   y=26〜44 (高さ18)
脚:   y=44〜47
全体: 50ユニット
```

### 新しい個別比率

**小人 (ずんぐり・低重心)**
```
帽子:  y=1〜28 (高さ27、体の60%を占める)
顔:    y=20〜33 (半径8)
体:    y=30〜43 (高さ13、横幅広め)
脚:    y=43〜47
```

**きつね (スリム・細長)**
```
耳:    y=2〜14 (高さ12、尖り)
頭:    r=10 (少し小さい)
体:    y=27〜43 (高さ16、横幅狭め)
尻尾:  巨大S字、体の1.5倍
脚:    y=43〜47 (細い)
```

**くま (丸・広幅)**
```
耳:    r=5 (大きい)
頭:    r=12.5 (一番大きい)
体:    y=26〜44 (高さ18、横幅広い)
脚:    y=44〜47 (短い)
マフラー: 幅広く流れる
```

**しか (細長・長脚)**
```
角:    y=1〜10 (縦に広がる枝)
頭:    r=9 (縦長楕円)
首:    y=20〜27 (細い)
体:    y=25〜41 (高さ16、横幅中)
脚:    y=41〜47 (長い、細い)
```

## 4. 顔パーツ設計

### 4-1. 目の工夫
```
サイズ:     r=1.3〜1.5（現状維持、小さめ）
色:        #3D3D3D ではなく少し温かい #3A3228 (ウォームブラウン)
ハイライト: 白い小さな丸（r=0.4）を右上に
配置:      キャラごとに間隔を変える
  - 小人:   離す (ひげ上の余白に)
  - きつね: 離す (マスク模様内)
  - くま:   近い (鼻が大きいため)
  - しか:   離す (横顔っぽく)
```

### 4-2. 鼻
```
色:        
  - 小人:   肌色の少し濃い (#C89878)
  - きつね: 黒小さい三角
  - くま:   黒大きい楕円
  - しか:   黒中くらい
形:         キャラの性格を反映
  - 小人は「団子鼻」
  - きつねは「鋭い三角」
  - くまは「丸い黒」
  - しかは「小さな縦長」
```

### 4-3. 口
```
基本:     口は表情の9割を決める
  - 小人:   ひげに隠れて見えない（これがキャラ）
  - きつね: 「>_<」の斜め笑い
  - くま:   大きな「ω」型
  - しか:   小さな「︵」型
```

### 4-4. 左右非対称の入れ方
```
- 片目を0.3ユニット下に
- 片耳を0.5ユニット前に
- 頬の位置を片側だけ少しずらす
- 脚の間隔を1ユニット左右非対称に
```

## 5. SVG制作ルール v2

### 5-1. 線の考え方
```
基本: 線なし (stroke="none")
例外: シルエットが読めない時のみ、薄い色で極細線
  stroke="#6B5A48"  (塗り色より少し濃い茶系)
  strokeWidth={0.4}
  opacity={0.5}
```

### 5-2. 塗りの3層化
```
Layer 1: BASE         フラットベース色 (100%)
Layer 2: SHADOW       暗色 (左下半分、opacity 1.0 で別パス)
Layer 3: HIGHLIGHT    明色 (右上の小さい円、opacity 0.5)
```

### 5-3. 色の階層
```
各キャラ3色ルール:
  Color1: BASE (メイン)
  Color2: DARK  (BASEの明度-15%, 彩度-5%)
  Color3: LIGHT (BASEの明度+10%, 彩度-10%)

※ DARK は別レイヤーで完全不透明
※ LIGHT はハイライト用、opacity 0.5
```

### 5-4. 目の描画ルール
```
<Circle cx={X} cy={Y} r={1.4} fill="#3A2818" />
<Circle cx={X + 0.4} cy={Y - 0.4} r={0.5} fill="#FFFFFF" />
```
→ 温かい黒 + 右上ハイライトで生命感

### 5-5. 接地影
```
2層:
  <Ellipse cx={20} cy={47.5} rx={12} ry={1.8} fill="#00000008" />
  <Ellipse cx={20} cy={47.8} rx={8} ry={1.2} fill="#00000012" />
```

# キャラクターデザイン改善仕様書

## 1. 現状の課題分析

### 1-1. シルエット問題
**現状**: 4体とも「丸い顔 + 丸い体」の構造が似通っていて、小さく表示した時に種類が判別しにくい
**原因**:
- 頭と体がほぼ同じ幅で、シルエットに変化がない
- 耳の大きさが控えめで、きつねもうさぎもくまも「円 + 小さな突起」に見える
- しっぽの存在感が薄い

### 1-2. 表情の弱さ
**現状**: 目が中央寄りにあり、左右対称で、表情がのっぺりしている
**原因**:
- 目の間隔が狭く、顔の白面積が余っている
- 鼻・口・眉の垂直位置が詰まっている
- 完璧な左右対称で「お面」のように見える

### 1-3. 生命感の不足
**現状**: 静止画のような佇まいで、ゲーム主人公としての勢いがない
**原因**:
- 四肢（腕・脚）がほぼ同じ形で棒のようにぶら下がっている
- パーツ配置に「間合い」「余白」がない
- 線の強弱がなく、全てが同じ主張

---

## 2. 改善方針

### コンセプト
**「小さく映っても誰だかわかる、強い主役シルエット」**

### 3本の軸
1. **シルエット差別化** — 各キャラを上半身のシルエットだけで見分けられるように
2. **顔の余白** — 白い面積を広げ、目の位置を外に寄せて親しみを増やす
3. **線と塗りの階層化** — 外輪郭 > 内部の線 > 影 の順に弱くして絵画的に

## 3. SVG制作ルール

### 3-1. 線幅の階層
```
外輪郭 (body, head silhouette):   1.2px
内側の主要線 (muzzle, ear inner): 0.8px
細部 (mouth, eyebrow, whiskers):   0.4〜0.5px
```
→ 外側が主張、内側はサポート。これで「絵」らしくなる。

### 3-2. 線色
```
メイン輪郭:    #5A5A5A (柔らかい濃グレー)
内部の線:      #6B6B6B (少し明るく)
目/鼻の黒:    #3D3D3D (本当の黒ではない)
```

### 3-3. 塗りの3段階
```
BASE:      メインの色
HIGHLIGHT: BASEの明度+15%、彩度-5%
SHADOW:    BASEの明度-15%、彩度-5%

※ HIGHLIGHT と SHADOW は opacity 0.35〜0.5 で重ねる
※ 影は常に「光源が右上から来る」想定で左下側に入れる
```

### 3-4. パーツ構成のルール
```
1体あたりのパス数: 20〜30個程度に抑える
目と口以外は閉じた Path で描く（線が切れない）
同じパーツ（左右の目、耳など）は対称コピーではなく微妙にずらす
```

### 3-5. 小さく表示した時の可読性
```
最小サイズ:   32x40 でもシルエットが判別できること
線の太さ:     viewBox は 40x50 で統一、線幅は絶対値 (scale無視)
細部の省略:   まつげ、爪、縫い目などは入れない
```

## 4. 頭身比率の統一

### 4体共通の新比率
```
全体高:         50ユニット
頭:             y=3 〜 y=30  (27ユニット = 54%)
首/肩:          y=28 〜 y=32 (体に埋没)
体:             y=28 〜 y=42 (14ユニット = 28%)
脚:             y=40 〜 y=47 (7ユニット = 14%)
影:             y=47 〜 y=50 (3ユニット = 6%)
```

### 各パーツの横幅
```
頭:       直径 22〜24
体:       幅 14〜16 (頭より明らかに細い)
脚:       幅 3〜4、太めで短い
腕:       幅 3、体から少し出る
```

→ 「頭が大きいが、体と脚で支えられている」安定したシルエット

## 5. キャラクター別仕様

### 5-1. 小人 (Kobito) — 主役中の主役

**シルエット特徴**: 三角の帽子（先端カール）+ 丸い顔 + 四角いマント

**比率**:
```
帽子:   y=2 〜 y=26  (高さ24 = シルエットの最大特徴)
顔:     y=15 〜 y=32 (直径18)  ※帽子と顔が重なる
体:     y=28 〜 y=42
脚:     y=40 〜 y=46
```

**印象的なポイント**:
- 帽子は画面上で最も目立つ要素 → 幅を体より広く（26 vs 16）
- 帽子の先端を右にカール（生命感）
- マントの裾が床近くで広がる（台形シルエット）
- 顔の下半分だけを見せる（帽子が深い）

**色**:
```
帽子:       #CC6055 (くすんだ赤)
帽子 HL:    #D87068
帽子 SH:    #A84A45
マント:     #7A9CAE (グレイッシュブルー)
マント HL:  #92B2C2
マント SH:  #5A7E90
肌:         #F0D8C0
靴:         #A08058
```

**顔の特徴**:
- 目は帽子の影の中にある → 暗い（少し強調）
- 白目の面積を最小限に → 黒めがち
- 鼻はごく小さい点 → 愛嬌
- 口は「にこっ」とした曲線

---

### 5-2. きつね (Kitsune) — 尻尾と耳が主役

**シルエット特徴**: 尖った大きな耳 + 白いマズル + **巨大な尻尾**

**比率**:
```
耳:     y=2 〜 y=11  (高さ9、根元の幅6)
頭:     y=8 〜 y=30  (直径22)
体:     y=28 〜 y=42
尻尾:   x=0 〜 x=12, y=25 〜 y=42 (画面幅の30%を占有)
脚:     y=40 〜 y=47
```

**印象的なポイント**:
- **尻尾を現状の2倍のボリューム** → ふさふさのSカーブで、画面左側を覆う
- **耳を大きく、根元太く、先端尖り** → 「きつね」と一目でわかる
- マズルを白く広く取り、鼻先を黒く強調
- 尻尾の先端は白
- 顔に「目の周りのオレンジ模様」を入れて顔に動きを

**色**:
```
毛 BASE:    #D8A060
毛 HL:      #E8B878
毛 SH:      #B88048
腹 / マズル: #F0E0C8
腹 SH:      #D8C8B0
鼻:         #3D3D3D
目:         #3D3D3D
```

**尻尾の構造**:
```
ベースカーブ: 下から上に大きくS字
内側:        白い差し色
先端:        白い tip
サイズ:      体の1.5倍の長さ
```

---

### 5-3. うさぎ (Usagi) — 長い垂れ耳

**シルエット特徴**: 長い垂れ耳 + 丸い顔 + 小さな丸い尻尾

**比率**:
```
左耳:   x=4 〜 x=15, y=5 〜 y=28 (ほぼ頭と同じ長さ)
右耳:   x=25 〜 x=36, y=5 〜 y=28
頭:     y=8 〜 y=28  (直径20)
体:     y=26 〜 y=42
尻尾:   x=8 〜 x=14, y=34 〜 y=38 (ふわっとした丸)
脚:     y=40 〜 y=47
```

**印象的なポイント**:
- **耳は頭と同じ長さ** → 下にだらんと垂れる、ロップイヤー
- 耳の内側はピンク（鮮やかすぎない）
- 鼻はY字のピンク
- ひげは繊細に（opacity 0.4）
- しっぽは球形でふわっと

**色**:
```
体 BASE:    #F0ECE8 (オフホワイト、真っ白にしない)
体 HL:      #FAF6F2
体 SH:      #D8D0C8
耳内側:     #E0C8C0
耳内 SH:    #C8A8A0
鼻:         #D0A098
```

**耳の形**:
```
左耳: 頭の左上から出て、左下にS字で垂れる
     根元: 幅5、先端: 幅3、形は少し丸みを帯びる
右耳: 左右対称ではなく、ごく僅かに違う角度で垂れる
```

---

### 5-4. くま (Kuma) — マフラーと丸い耳

**シルエット特徴**: **小さな丸耳** + マフラー + ずんぐりした体

**比率**:
```
左耳:   円、中心(7, 6)、半径4
右耳:   円、中心(33, 6)、半径4
頭:     y=6 〜 y=28 (直径22)
マフラー: y=25 〜 y=32 (首周り)、先端が画面左下に垂れる
体:     y=28 〜 y=42
脚:     y=40 〜 y=47
```

**印象的なポイント**:
- **耳は完全な円** → 「くま」とすぐわかる最強の記号
- 耳の中を明色で描く
- マフラーの先端を斜めに伸ばして動きを出す
- マズル（口周り）が明色で広め
- 目の間が狭め（くまの特徴）

**色**:
```
毛 BASE:    #B08050
毛 HL:      #C69868
毛 SH:      #8E6238
腹 / マズル: #D8C0A0
腹 SH:      #B8A080
マフラー:   #CC6055
マフラー SH:#A84A45
```

## 6. 顔のパーツ配置ルール（4体共通）

### 目の位置
```
頭の中心から左右に幅の30% 離す
→ 現状の25%より外側
→ 顔の真ん中に余白ができ、鼻・口のスペースが生まれる
```

### 目のサイズ
```
白目の楕円: rx=1.6, ry=2.0 (やや縦長)
黒目:      r=1.4
ハイライト: r=0.6 (右上)
→ 目の上部を少し欠けさせると「笑顔」の印象が出る
```

### 眉
```
目の上に短い弧
線幅 0.4、opacity 0.4
左右で微妙に高さを変える（1単位ずらす）
```

### 鼻と口の距離
```
鼻: 目の下3ユニット
口: 鼻の下2ユニット
→ 詰めすぎず、離しすぎず
```

### 左右非対称の入れ方
```
- 右目を左目より0.3ユニット大きく
- 口の右端を左端より0.2ユニット上に
- 頬の位置を片方だけ少しずらす
→ 「生きている」感じに
```

## 7. 共通の改善項目

### 7-1. 接地影
```
現状: Ellipse cx=20 cy=48 rx=10 ry=2.5 opacity=0.04
改善: Ellipse cx=20 cy=47 rx=11 ry=2 + 内側にもう1枚 rx=7 ry=1.3
→ 2段重ねで自然な影に
```

### 7-2. 脚の太さ
```
現状: 幅3ユニット
改善: 幅4ユニット、下部は少し太く（台形）
→ 安定感、絵本らしさ
```

### 7-3. 腕の処理
```
現状: 真横から出る小さな突起
改善: 体の側面から少し下に向かって出る（自然な位置）
     長さを少し長く、先端を丸く
```

### 7-4. 体のシルエット
```
現状: ほぼ長方形
改善: 上が細く下が広がる台形（安定感）
     または、上が広く下が狭い樽型（親しみ）
※ キャラごとに使い分ける
  - 小人: 台形 (マントが広がる)
  - きつね: 樽型
  - うさぎ: 樽型
  - くま: 樽型（一番ずんぐり）
```

## 8. 実装の優先順位

### Phase A: シルエット強化（最大のインパクト）
1. きつねの尻尾を2倍に、S字カーブで左に大きく張り出し
2. うさぎの耳を頭と同じ長さに、S字で垂れる
3. くまの耳を完璧な円に
4. 小人の帽子の先端にカール

### Phase B: 顔の配置
5. 目を30%外側にずらす
6. 白目を縦長楕円に
7. 眉を追加（左右非対称）
8. 鼻・口の間隔を整える

### Phase C: 線の階層化
9. 外輪郭を 1.2px に
10. 内部線を 0.8px に
11. 細部線を 0.4〜0.5px に

### Phase D: 塗りの階層化
12. 全キャラに HIGHLIGHT / SHADOW レイヤーを追加（各 opacity 0.35〜0.5）
13. 光源を右上固定

### Phase E: 細部の生命感
14. 左右非対称の微調整
15. 接地影の2段重ね
16. 脚の台形化

## 10. 注意事項

- **幼児っぽくしない** — 目を大きくしすぎない、頬のチークを強くしすぎない
- **シリアスにしない** — 眉を強くしすぎない、黒を使いすぎない
- **記号化しすぎない** — 動物の特徴は残すが、デフォルメで省略
- **現状の可愛さを壊さない** — 大幅な方向転換ではなく「磨き上げ」
- **小さい表示でテストする** — 実機で 50px サイズで見て判別できるか確認

これでPhase順に実装していけば、現状のキャラが絵本の主役として通用するレベルに上がる。
